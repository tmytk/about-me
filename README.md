# takahashi.dev

Tomoya Takahashiのポートフォリオサイトです。AstroのContent Collectionsで制作物、技術メモ、日々のメモ、音楽ログを管理しています。

## Stack

- Astro
- TypeScript
- Markdown / MDX-style content collections
- KaTeX for math rendering

## Commands

```sh
npm run dev
npm run build
npm run preview
```

## Content

Markdownは `src/content` 配下に配置します。

- `src/content/works`: 制作物と技術メモ
- `src/content/notes`: 日々のメモ
- `src/content/music`: 音楽ログ
